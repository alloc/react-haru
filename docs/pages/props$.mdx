# Reserved props

Since the following keys are reserved for features, they won't be inferred as `to` values unless you nest them in a `to` object.

- cancel
- config
- default
- delay
- from
- immediate
- loop
- pause
- onChange
- onPause
- onProps
- onResolve
- onRest
- onResume
- onStart
- ref
- reset
- reverse
- to
